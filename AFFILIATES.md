# Affiliate Programs

This document describes each affiliate/referral program used by TheresAlwaysADeal, including the environment variable name, query parameter format, and current program status.

## Steam

- **Environment Variable:** `STEAM_AFFILIATE_TAG`
- **Query Parameter:** `?partner=<tag>` (appended to store.steampowered.com URLs)
- **Status:** Active — Steam Partner program via Xsolla

## GOG

- **Environment Variable:** `GOG_AFFILIATE_ID`
- **Query Parameter:** `?affiliate_id=<id>` (appended to gog.com URLs)
- **Status:** Active — GOG Affiliate program

## Epic Games

- **Environment Variable:** `EPIC_CREATOR_TAG`
- **Query Parameter:** `?epicCreatorId=<tag>` (appended to store.epicgames.com URLs)
- **Status:** Active — Epic Games Support-a-Creator program

## Humble Bundle

- **Environment Variable:** `HUMBLE_PARTNER_ID`
- **Query Parameter:** `?partner=<id>` (appended to humblebundle.com URLs)
- **Status:** Active — Humble Partner program

## Fanatical

- **Environment Variable:** `FANATICAL_REF`
- **Query Parameter:** `?ref=<ref>` (appended to fanatical.com URLs)
- **Status:** Active — Fanatical Affiliate program
